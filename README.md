# Random maze generation
Random maze generation using the algorithm Recursive backtracker. To learn more about the algorithm, follow the link -> https://habr.com/ru/articles/1002460/

We create a labyrinth without loops or closed circuits, without inaccessible areas, and with exactly one path from each point to every other point. The resulting labyrinth is defined by a set of zeros and ones, where zero is a passage and one is a wall.
